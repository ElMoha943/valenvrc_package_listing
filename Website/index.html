How to Add This Listing to Your VCC

To add this listing to your VCC (VRChat Creator Companion):

  1. Open your VCC and go to Settings
  2. Click the Packages tab
  3. Click Add Repository
  4. Paste the URL displayed above
  5. Click Add
  6. Review the repository info and click I Understand
  7. Go to any project to see packages from this listing
